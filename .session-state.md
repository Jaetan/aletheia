### Semantic Contract

The monitor follows **standard RV-LTL semantics** (Bauer-Leucker-Schallhart 2011):
- Safety operators (Always, Release) **propagate Inconclusive** (no premature violation)
- Liveness operators (Eventually, Until) return Inconclusive on Unknown
- Pending atoms trigger retry (Continue) rather than immediate verdict
- **No false positives or false negatives** — monitor only gives definite answers when correct

The relationship between `runL` and `⟦_⟧` is **bilateral Sound** (no wrong definite answers in either direction):
- `Sound True True` — definite agreement
- `Sound False False` — definite agreement
- `Sound m Unknown`, `Sound m Pending` — denotation uncertain, any monitor result OK
- `Sound Unknown d`, `Sound Pending d` — monitor admits ignorance
- NOT `Sound True False`, NOT `Sound False True` — never wrong definite answers

### Proof Architecture Summary

Four-layer proof structure (1061 lines, Adequacy.agda):

1. **Sound compositionality** (`sound-and`, `sound-or`, `sound-not`): compose IH through Kleene connectives
2. **Operational decomposition** (`runL-*-decomp`): `runL table (Op φ ψ) σ ≡ runL table φ σ ⊕TV runL table ψ σ`
3. **Soundness transport** (`sound-transport-monitor`, `sound-transport-denot`): `subst`-based rewriting without `with`-auxiliaries
4. **Non-recursive metric helpers** (`adequacy-met-*`): extract `with` case analysis to fix termination checker

### Key Lessons Learned

- **Nested `with` on two `stepL` results** breaks `combineOr`/`combineAnd` reduction (And-vs-Or issue)
- **Simultaneous `with stepL ψ y | stepL φ y`** works — both values concrete after match, no abstract intermediates
- **`rewrite` in recursive functions** generates `with`-auxiliaries that hide recursive calls from termination checker
- **Non-recursive helpers** with pre-computed IH arguments solve the termination issue
- **`subst`/`sound-transport-*`** avoids `with`-auxiliaries entirely (function application, not pattern matching)
- **Decomposition lemmas inside `with boolean | true`** don't work (normal form mismatch from with-abstraction)
