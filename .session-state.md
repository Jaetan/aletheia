## Key Learnings

- **Stale .so**: When Agda data type constructors change, use clean build
- **formatRational int vs dict**: Agda outputs plain integer when denominator=1; Python `_to_rational()` handles both
- **`with` inside `let`**: Invalid Agda syntax — use `where` block helpers
- **Bisimilarity implicit patterns**: Adding implicit params to `Relate` constructors shifts positional bindings — fix with `{_}` skip
- **`finalize-bisim`**: Requires same `Bool` resolved flag on both `LTLEvalState` and `LTLProc` constructors so proof works by `refl`
